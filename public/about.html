About splash-js

splash-js is a program to edit lenex files. Lenex files are used to enter swimming competitions.

The program is written in JavaScript and uses HTML5. It uses some third party JavaScript libraries.

The main audience is the Lifesaving competitions in the Netherlands, but other swimming competitions should also be able to use the software.
